# Анимации в проекте EC Курсы

- **fadeInUp** - плавное появление снизу вверх

## Используемые классы CSS

### Анимации:
- `.animate-fade-in-up` - появление снизу
- `.animate-fade-in` - простое появление
- `.animate-scale-in` - появление с увеличением
- `.animate-slide-in-left` - скольжение слева
- `.animate-slide-in-right` - скольжение справа
- `.animate-pulse-slow` - медленная пульсация

### Задержки:
- `.delay-100`, `.delay-200`, `.delay-300`, `.delay-400`, `.delay-500`

## Как использовать

### В CSS:
```css
.my-element {
  @apply animate-fade-in-up;
}
```

### В React компонентах с Framer Motion:
```tsx
import { motion } from "motion/react";

<motion.div
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.5, delay: 0.2 }}
>
  Содержимое
</motion.div>
```

### Эффекты при взаимодействии:
```tsx
<motion.div
  whileHover={{ scale: 1.05, y: -8 }}
  whileTap={{ scale: 0.95 }}
>
  Интерактивный элемент
</motion.div>
```

## Параметры анимаций

- **duration** - длительность анимации в секундах
- **delay** - задержка перед началом
- **ease** - тип плавности (ease-out, ease-in, ease-in-out)
- **whileHover** - анимация при наведении
- **whileTap** - анимация при клике

## Рекомендуемые практики

1. Используйте задержки (delay) для последовательных анимаций
2. Не злоупотребляйте анимациями - 0.3-0.6 сек оптимально
3. Для интерактивных элементов используйте `whileHover` и `whileTap`
4. При переходе между страницами используйте `initial` и `animate` для плавности
